# page-schema

Objects in the page schema are used to define the structure of a page. They are used to define the information, content and components that are used to render the page.

import { Code } from 'astro:components'
import { zodToTs, createTypeAlias, printNode } from 'zod-to-ts'
import pageSchema from 'fulldev-ui/schemas/pageSchema'

export const identifier = 'Page'
export const node = zodToTs(pageSchema, identifier)
export const typeAlias = createTypeAlias(node, identifier)
