# Installation

How to install dependencies and setup the integration.

## Add to existing project

If you don't have an existing project, you can create a new one using the [starter template](https://github.com/fulldotdev/starter).

### Automatic setup

Run the following command to automatically install the dependencies and add the integration.

```bash
npx astro add fulldev-ui
```

### Manual setup

Run the following command to add fulldev-ui to your dependencies.

```bash
npm i fulldev-ui
```

Add the integration to your `astro.config.ts` file.

```ts
import { defineConfig } from 'astro/config'
import fulldev from 'fulldev-ui/integration'

export default defineConfig({
  integrations: [
    fulldev({
      // Add any options here
    }),
  ],
})
```

## Create new project

Run the following command to setup a new project from the [starter template](https://github.com/fulldotdev/starter).

```bash
npx astro create --template fulldev-ui
```
