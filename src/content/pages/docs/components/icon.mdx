Lucide ships Astro components directly, so you can import only the icons you
need and render them like any other Astro component.

```astro live
---
import { Check, Heart, MessageCircle, Star, Trash2 } from "@lucide/astro"
---

<Heart />
<Star />
<Check />
<Trash2 />
<MessageCircle />
```

## Installation

Install Lucide's Astro package:

```bash
bun add @lucide/astro
```

## Usage

```ts
import { Heart } from "@lucide/astro"
```

```astro
<Heart />
```

## Example

Inside buttons and similar controls, use `data-icon` so the surrounding
component can size and align the icon consistently.

```astro live
---
import { ArrowRight } from "@lucide/astro"

import { Button } from "@/components/ui/button"
---

<Button>
  Continue
  <ArrowRight data-icon="inline-end" />
</Button>
```

## Notes

- Import only the icons you need.
- Prefer passing icon components directly instead of storing string keys in
  content or config.
