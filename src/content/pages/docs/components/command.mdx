## About

The `Command` components in this repo match the visual styling of the shadcn/ui
command menu, while using `@data-slot/command` for filtering, ranking, keyboard
navigation, and selection.

## Installation

### Command

This component is not yet available as a published `@fulldev/*` registry item.
Use the manual installation steps below.

### Manual

1. Install the dependency:

```bash
npm install @data-slot/command
```

2. Copy the files from `src/components/ui/command/` into your project.

3. Update the import paths to match your project setup.

## Composition

Use the following composition to build a `Command`:

```text
Command
├── CommandInput
└── CommandList
    ├── CommandEmpty
    ├── CommandGroup
    │   ├── CommandItem
    │   └── CommandItem
    ├── CommandSeparator
    └── CommandGroup
        ├── CommandItem
        └── CommandItem
```

### Scrollable

Scrollable command menu with multiple items.

```astro live
---
import { ClockIcon } from "@lucide/astro"

import {
  Command,
  CommandEmpty,
  CommandGroup,
  CommandInput,
  CommandItem,
  CommandList,
} from "@/components/ui/command"

const recentSearches = Array.from({ length: 18 }, (_, index) => ({
  label: `Recent search ${index + 1}`,
  value: `recent-search-${index + 1}`,
}))
---

<Command class="ring-foreground/10 h-[24.5rem] w-full max-w-md ring-1">
  <CommandInput placeholder="Search recent activity..." />
  <CommandList>
    <CommandEmpty>No results found.</CommandEmpty>
    <CommandGroup heading="Recent">
      {
        recentSearches.map((item) => (
          <CommandItem value={item.value}>
            <ClockIcon />
            {item.label}
          </CommandItem>
        ))
      }
    </CommandGroup>
  </CommandList>
</Command>
```

mode.
