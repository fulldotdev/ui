## Installation

### Command

```bash
npx shadcn@latest add switch
```

### Manual

1. Install the following dependency:

```bash
npm install @data-slot/switch
```

2. Copy and paste the switch component files into `src/components/ui/switch/`.

3. Update the import paths to match your project setup.

## Usage

```astro
---
import { Switch, SwitchThumb } from "@/components/ui/switch"
---
```

```astro
<Switch>
  <SwitchThumb />
</Switch>
```

### Disabled

Add the `disabled` prop to the `Switch` component to disable the switch. Add
the `data-disabled` prop to the `Field` component for styling.

```astro live
---
import {
  Field,
  FieldContent,
  FieldDescription,
  FieldLabel,
} from "@/components/ui/field"
import { Switch, SwitchThumb } from "@/components/ui/switch"
---

<div class="w-full max-w-sm">
  <Field orientation="horizontal" data-disabled="true">
    <Switch id="security-alerts" name="security-alerts" disabled>
      <SwitchThumb />
    </Switch>
    <FieldContent>
      <FieldLabel for="security-alerts">Security alerts</FieldLabel>
      <FieldDescription>
        This setting is managed by your organization.
      </FieldDescription>
    </FieldContent>
  </Field>
</div>
```

### Invalid

Add the `aria-invalid` prop to the `Switch` component to indicate an invalid
state. Add the `data-invalid` prop to the `Field` component for styling.

```astro live
---
import {
  Field,
  FieldContent,
  FieldDescription,
  FieldLabel,
} from "@/components/ui/field"
import { Switch, SwitchThumb } from "@/components/ui/switch"
---

<div class="w-full max-w-sm">
  <Field orientation="horizontal" data-invalid="true">
    <Switch id="terms-switch" name="terms-switch" required aria-invalid="true">
      <SwitchThumb />
    </Switch>
    <FieldContent>
      <FieldLabel for="terms-switch">Accept terms</FieldLabel>
      <FieldDescription class="text-destructive">
        You must enable this switch before continuing.
      </FieldDescription>
    </FieldContent>
  </Field>
</div>
```

### Size

Use the `size` prop to change the size of the switch.

```astro live
---
import { Label } from "@/components/ui/label"
import { Switch, SwitchThumb } from "@/components/ui/switch"
---

<div class="flex flex-col gap-4">
  <Label class="inline-flex items-center gap-3">
    <Switch size="sm" name="switch-size-sm">
      <SwitchThumb />
    </Switch>
    Small
  </Label>
  <Label class="inline-flex items-center gap-3">
    <Switch size="default" name="switch-size-default" defaultChecked>
      <SwitchThumb />
    </Switch>
    Default
  </Label>
</div>
```

## RTL

To enable RTL support, apply `dir="rtl"` to a parent container.

```astro live
---
import { Label } from "@/components/ui/label"
import { Switch, SwitchThumb } from "@/components/ui/switch"
---

<div dir="rtl">
  <Label class="inline-flex items-center gap-3">
    <Switch name="rtl-switch" defaultChecked>
      <SwitchThumb />
    </Switch>
    Notifications
  </Label>
</div>
```
