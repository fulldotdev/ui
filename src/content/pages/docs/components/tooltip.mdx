---
type: doc
title: Tooltip
description: Displays contextual information when a user hovers or focuses a trigger.
---

```astro live
---
import { buttonVariants } from "@/components/ui/button"
import {
  Tooltip,
  TooltipArrow,
  TooltipContent,
  TooltipTrigger,
} from "@/components/ui/tooltip"
---

<Tooltip>
  <TooltipTrigger class={buttonVariants({ variant: "outline" })}>
    Hover me
  </TooltipTrigger>
  <TooltipContent>
    Helpful tooltip text
    <TooltipArrow />
  </TooltipContent>
</Tooltip>
```

## Installation

To install the tooltip component, run the following command:

```bash
npx shadcn@latest add @fulldev/tooltip
```

The component is powered by `@data-slot/tooltip`.

## Usage

```ts
import {
  Tooltip,
  TooltipArrow,
  TooltipContent,
  TooltipPortal,
  TooltipPositioner,
  TooltipTrigger,
} from "@/components/ui/tooltip"
```

```astro
<Tooltip>
  <TooltipTrigger class="rounded-md border px-3 py-2">Hover me</TooltipTrigger>
  <TooltipContent>
    Helpful tooltip text
    <TooltipArrow />
  </TooltipContent>
</Tooltip>
```

### Faster Delay

```astro live

  Tooltip,

  TooltipContent,
```
