## Installation

```bash
npm install @data-slot/slider
```

## Usage

```astro
---
import {
  Slider,
  SliderControl,
  SliderRange,
  SliderThumb,
  SliderTrack,
} from "@/components/ui/slider"
---
```

```astro
<Slider defaultValue={33} max={100} step={1}>
  <SliderControl>
    <SliderTrack>
      <SliderRange />
    </SliderTrack>
    <SliderThumb />
  </SliderControl>
</Slider>
```

This Astro wrapper keeps the authored slot structure from `@data-slot/slider`,
so `SliderControl`, `SliderTrack`, `SliderRange`, and `SliderThumb` remain
explicit in markup.

## Examples

### Range

Use an array with two values for a range slider.

```astro live
---
import {
  Slider,
  SliderControl,
  SliderRange,
  SliderThumb,
  SliderTrack,
} from "@/components/ui/slider"
---

<Slider
  defaultValue={[25, 75]}
  max={100}
  step={1}
  thumbAlignment="edge"
  class="w-full max-w-md"
>
  <SliderControl>
    <SliderTrack>
      <SliderRange />
    </SliderTrack>
    <SliderThumb label="Minimum value" />
    <SliderThumb label="Maximum value" />
  </SliderControl>
</Slider>
```

### Controlled

This wrapper does not expose a React-style controlled prop API. Use the runtime
events from `@data-slot/slider` for programmatic control.

```astro live
---
import { Button } from "@/components/ui/button"
import {
  Slider,
  SliderControl,
  SliderRange,
  SliderThumb,
  SliderTrack,
} from "@/components/ui/slider"
---

<div class="w-full max-w-md space-y-4" data-demo-slider-controlled>
  <Slider
    defaultValue={40}
    max={100}
    step={10}
    thumbAlignment="edge"
    data-demo-slider-controlled-root
  >
    <SliderControl>
      <SliderTrack>
        <SliderRange />
      </SliderTrack>
      <SliderThumb />
    </SliderControl>
  </Slider>

  <div class="flex flex-wrap gap-2">
    <Button data-demo-set-value="20" size="sm" variant="outline">20</Button>
    <Button data-demo-set-value="50" size="sm" variant="outline">50</Button>
    <Button data-demo-set-value="80" size="sm" variant="outline">80</Button>
  </div>

  <p class="text-muted-foreground text-sm" data-demo-slider-value>Value: 40</p>
</div>

<script>
  const demo = document.querySelector("[data-demo-slider-controlled]")
  const root = demo?.querySelector("[data-demo-slider-controlled-root]")
  const output = demo?.querySelector("[data-demo-slider-value]")
  const buttons = demo?.querySelectorAll("[data-demo-set-value]") ?? []

  const renderValue = (value) => {
    const label = Array.isArray(value) ? value.join(", ") : value
    if (output) output.textContent = `Value: ${label}`
  }

  root?.addEventListener("slider:change", (event) => {
    renderValue(event.detail.value)
  })

  buttons.forEach((button) => {
    button.addEventListener("click", () => {
      const value = Number(button.getAttribute("data-demo-set-value"))

      root?.dispatchEvent(
        new CustomEvent("slider:set", {
          detail: { value },
        })
      )
    })
  })
</script>
```

### Disabled

Use the `disabled` prop to disable the slider.

```astro live
---
import {
  Slider,
  SliderControl,
  SliderRange,
  SliderThumb,
  SliderTrack,
} from "@/components/ui/slider"
---

<Slider
  defaultValue={50}
  disabled
  thumbAlignment="edge"
  class="w-full max-w-md"
>
  <SliderControl>
    <SliderTrack>
      <SliderRange />
    </SliderTrack>
    <SliderThumb />
  </SliderControl>
</Slider>
```
