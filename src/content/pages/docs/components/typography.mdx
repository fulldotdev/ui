## Installation

To install the typography component, run the following command:

```bash
npx shadcn@latest add @fulldev/typography
```

## Usage

```ts
import {
  Typography,
  TypographyA,
  TypographyBlockquote,
  TypographyH1,
  TypographyH2,
  TypographyH3,
  TypographyH4,
  TypographyInlineCode,
  TypographyLarge,
  TypographyLead,
  TypographyList,
  TypographyListItem,
  TypographyMuted,
  TypographyP,
  TypographySmall,
} from "@/components/ui/typography"
```
