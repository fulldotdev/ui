```astro live

import { Button } from "@/components/ui/button"

---

<Section>
  <SectionContent>
    <h2 class="text-2xl font-semibold">Section heading</h2>
    <p>This is a section component for organizing page content.</p>
    <div class="flex flex-wrap gap-2">
      <Button>Action</Button>
      <Button variant="outline">Action</Button>
    </div>
  </SectionContent>
</Section>
```

## Installation

```bash
npx shadcn@latest add @fulldev/section
```

## Usage

```ts
import { Section, SectionContent } from "@/components/ui/section"
```

```astro
<Section>
  <SectionContent>
    <h2 class="text-2xl font-semibold">Section heading</h2>
    <p>Section content goes here.</p>
  </SectionContent>
</Section>
```

## Examples

### Floating

```astro live
---
import { Section, SectionContent } from "@/components/ui/section"
---

<Section variant="floating">
  <SectionContent>
    <h2 class="text-2xl font-semibold">Section heading</h2>
    <p>This is a section component for organizing page content.</p>
  </SectionContent>
</Section>
```

## Notes

- `Section` controls spacing and surface treatment.
- `SectionContent` keeps the inner width and horizontal padding consistent.
- Use `class` on `Section` when a specific page needs custom vertical spacing.
