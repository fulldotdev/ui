## Installation

To install the banner component, run the following command:

```bash
npx shadcn@latest add @fulldev/banner
```

## Usage

```ts
import {
  Banner,
  BannerClose,
  BannerContent,
  BannerDescription,
  BannerTitle,
} from "@/components/ui/banner"
```

```astro
<Banner>
  <BannerContent>
    <BannerTitle>Banner Title</BannerTitle>
    <BannerDescription>Banner Description</BannerDescription>
  </BannerContent>
</Banner>
```

`BannerContent` renders a default `BannerClose` for the standard dismiss pattern.
Set `showClose={false}` when you want to place your own `BannerClose`.

```astro
<Banner>
  <BannerContent showClose={false}>
    <BannerTitle>Banner Title</BannerTitle>
    <BannerDescription>Banner Description</BannerDescription>
    <BannerClose aria-label="Dismiss banner" />
  </BannerContent>
</Banner>
```
