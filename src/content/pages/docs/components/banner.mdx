## Installation

```bash
npx shadcn@latest add @fulldev/banner
```

## Usage

```ts
import {
  Banner,
  BannerClose,
  BannerContent,
  BannerDescription,
  BannerTitle,
} from "@/components/ui/banner"
```

```astro
<Banner>
  <BannerContent>
    <BannerTitle>Banner Title</BannerTitle>
    <BannerDescription>Banner Description</BannerDescription>
  </BannerContent>
</Banner>
```

`BannerContent` renders a `BannerClose` by default. Set `showClose={false}` if
you want to place your own close button.

```astro
<Banner>
  <BannerContent showClose={false}>
    <BannerTitle>Banner Title</BannerTitle>
    <BannerDescription>Banner Description</BannerDescription>
    <BannerClose aria-label="Dismiss banner" />
  </BannerContent>
</Banner>
```

## Examples

### Default

```astro live
---
import {
  Banner,
  BannerContent,
  BannerDescription,
  BannerTitle,
} from "@/components/ui/banner"
---

<Banner>
  <BannerContent>
    <BannerTitle>New docs are live</BannerTitle>
    <BannerDescription>
      Browse the latest component examples and installation guides.
    </BannerDescription>
  </BannerContent>
</Banner>
```

### Floating

```astro live
---
import {
  Banner,
  BannerContent,
  BannerDescription,
  BannerTitle,
} from "@/components/ui/banner"
---

<Banner variant="floating" storageKey="docs-floating-banner">
  <BannerContent>
    <BannerTitle>Summer launch</BannerTitle>
    <BannerDescription>
      Floating banners work well for announcements that sit above page content.
    </BannerDescription>
  </BannerContent>
</Banner>
```

## Notes

- Use `variant="default"` for full-width announcements.
- Use `variant="floating"` for inset announcement bars.
- Pass `storageKey` when a dismissed banner should stay hidden for the current browser session.
