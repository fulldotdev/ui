## Installation

### Command

```bash
npx shadcn@latest add @fulldev/dropdown-menu
```

### Manual

1. Install the following dependency:

```bash
npm install @data-slot/dropdown-menu
```

2. Copy the dropdown menu files from `src/components/ui/dropdown-menu/` into
   your project.

3. Update the import paths to match your project setup.

## Usage

```astro
---
import ChevronDownIcon from "lucide-static/icons/chevron-down.svg"

import { buttonVariants } from "@/components/ui/button"
import {
  DropdownMenu,
  DropdownMenuCheckboxItem,
  DropdownMenuContent,
  DropdownMenuGroup,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuPortal,
  DropdownMenuPositioner,
  DropdownMenuRadioItem,
  DropdownMenuSeparator,
  DropdownMenuShortcut,
  DropdownMenuTrigger,
} from "@/components/ui/dropdown-menu"
---
```

```astro
<DropdownMenu>
  <DropdownMenuTrigger class={buttonVariants({ variant: "outline" })}>
    Open
    <ChevronDownIcon class="size-4 opacity-60" />
  </DropdownMenuTrigger>
  <DropdownMenuContent class="w-56">
    <DropdownMenuGroup>
      <DropdownMenuLabel>My Account</DropdownMenuLabel>
      <DropdownMenuItem>Profile</DropdownMenuItem>
      <DropdownMenuItem>Billing</DropdownMenuItem>
    </DropdownMenuGroup>
    <DropdownMenuSeparator />
    <DropdownMenuGroup>
      <DropdownMenuCheckboxItem value="email">
        Email notifications
      </DropdownMenuCheckboxItem>
      <DropdownMenuRadioItem value="pro">Pro plan</DropdownMenuRadioItem>
    </DropdownMenuGroup>
  </DropdownMenuContent>
</DropdownMenu>
```

The Astro wrapper mirrors the data-slot authoring model. Use
`DropdownMenuItem` for action items, `DropdownMenuRadioItem` for single-select
items, and `DropdownMenuCheckboxItem` for multi-select items.

## Composition

Use the following composition to build a `DropdownMenu`:

```text
DropdownMenu
├── DropdownMenuTrigger
└── DropdownMenuContent
    ├── DropdownMenuGroup
    │   ├── DropdownMenuLabel
    │   ├── DropdownMenuItem
    │   └── DropdownMenuItem
    ├── DropdownMenuSeparator
    ├── DropdownMenuGroup
    │   ├── DropdownMenuCheckboxItem
    │   └── DropdownMenuCheckboxItem
    ├── DropdownMenuSeparator
    └── DropdownMenuGroup
        ├── DropdownMenuRadioItem
        └── DropdownMenuRadioItem
```

For manual portal composition:

```text
DropdownMenu
├── DropdownMenuTrigger
└── DropdownMenuPortal
    └── DropdownMenuPositioner
        └── DropdownMenuContent
```

## Runtime API

The Astro wrapper auto-discovers dropdown menu roots with `create()`, so you
can still use the `@data-slot/dropdown-menu` runtime events directly on the DOM
node.

```html
<div data-slot="dropdown-menu" id="plan-menu">...</div>

<script>
  const root = document.querySelector("#plan-menu")

  root?.addEventListener("dropdown-menu:value-change", (event) => {
    console.log(event.detail.value)
  })

  root?.dispatchEvent(
    new CustomEvent("dropdown-menu:set", {
      detail: { value: "pro", open: true },
    })
  )
</script>
```

## RTL

The underlying runtime supports directional layout. Apply `dir="rtl"` to a
parent container to render the menu in right-to-left mode.

## API Reference

See the `@data-slot/dropdown-menu` docs in your project for the full runtime
contract, including controller methods, events, and data attributes.
