# Icon

The Icon component uses [astro-icon](https://www.astroicon.dev/) under the hood. [Tabler Icons](https://tablericons.com/) are pre-installed and used as the default icon set.

```astro live
---
import Icon from 'fulldev-ui/components/Icon.astro'
---

<Icon name="confetti" />
```

## Examples

### `name`

```astro live
---
import Icon from 'fulldev-ui/components/Icon.astro'
---

<Icon name="confetti" />
<Icon name="brand-x" />
<Icon name="brand-github" />
```

### `size`

```astro live
---
import Icon from 'fulldev-ui/components/Icon.astro'
---

<Icon name="confetti" size="sm" />
<Icon name="confetti" size="md" />
<Icon name="confetti" size="lg" />
```

### `set`

To use a different icon set, it should be installed according to [these steps.](https://www.astroicon.dev/guides/customization/#open-source-icon-sets)

```astro
---
import Icon from 'fulldev-ui/components/Icon.astro'
---

<Icon set="lucide" name="circle" />
```
