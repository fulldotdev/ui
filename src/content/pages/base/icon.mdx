# Icon

We use [Tabler Icons](https://tablericons.com/) under the hood, you can use any of those icons by having the `name` prop match one of the icon names.

```astro live
---
import Icon from 'fulldev-ui/components/Icon.astro'
---

<Icon name="code" color="brand" size="lg" />
```
