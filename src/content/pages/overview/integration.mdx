# Integration

After <Link href="/overview/installation">installation</Link> you can generate your color palettes via an Astro integration. You can optionally add a custom css file and define custom colors and other theming options, see the <Link href="/overview/theming">theming page</Link> for more information.

```ts
import { defineConfig } from 'astro/config'
import fulldev from 'fulldev-ui/integration'

// https://astro.build/config
export default defineConfig({
  integrations: [
    fulldev({
      css: '/src/css/custom.css',
      colors: {
        theme: 'dark',
        dark: {
          background: '#111110',
          base: '#6F6D66',
          brand: '#F50',
        },
        light: {
          background: '#EEEEEC',
          base: '#6F6D66',
          brand: '#F50',
        },
      },
    }),
  ],
})
```
