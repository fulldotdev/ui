# Installation

Install fulldev-ui with your favorite package manager and you are ready to go. To get started with the integration see the <Link href="/overview/integration">integration page.</Link>

```bash
pnpm i fulldev-ui
# or
bun i fulldev-ui
# or
yarn add fulldev-ui
# or
npm i fulldev-ui
```
