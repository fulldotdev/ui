# Installation

Install fulldev-ui with your favorite package manager and you are ready to go.

```bash
pnpm i fulldev-ui
# or
bun i fulldev-ui
# or
yarn add fulldev-ui
# or
npm i fulldev-ui
```

## Integration

To install the integration, import it in your `astro.config.ts` file.
You can optionally add a custom css file and define custom colors and other theming options, see the <Link href="/overview/customization/">customization documentation</Link> for more information.

To get started with the content layer, see the <Link href="/overview/content-layer/#manual-setup">content layer documentation</Link>.

```ts
import { defineConfig } from 'astro/config'
import fulldev from 'fulldev-ui/integration'

// https://astro.build/config
export default defineConfig({
  integrations: [
    fulldev({
      css: '/src/css/custom.css',
      colors: {
        theme: 'dark',
        dark: {
          background: '#111110',
          base: '#6F6D66',
          brand: '#F50',
        },
        light: {
          background: '#EEEEEC',
          base: '#6F6D66',
          brand: '#F50',
        },
      },
    }),
  ],
})
```
