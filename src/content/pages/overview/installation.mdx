# Installation

Using astro's cli you can easily install the library and get a basic setup of the integration.

```bash
npx astro add fulldev-ui
```

## Starter template

The easiest way to get started is to use the <Link href="https://github.com/fulldotdev/starter">starter template</Link>.

This allows you to create websites using the content layer and the integration.

```bash
pnpm create astro@latest -- --template fulldotdev/starter
```

For more information see the <Link href="/overview/content-collections/">content collections documentation</Link>.

## Manual installation

Install fulldev-ui with your favorite package manager.

```bash
pnpm i fulldev-ui
# or
bun i fulldev-ui
# or
yarn add fulldev-ui
# or
npm i fulldev-ui
```

### Integration

To install the integration, import it in your `astro.config.ts` file.
You can optionally add a custom css file and define custom colors and other theming options, see the <Link href="/overview/customization/">customization documentation</Link> for more information.

To get started with the content layer, see the <Link href="/overview/content-layer/#manual-setup">content layer documentation</Link>.

```ts
import { defineConfig } from 'astro/config'
import fulldev from 'fulldev-ui/integration'

// https://astro.build/config
export default defineConfig({
  integrations: [
    fulldev({
      // options here
    }),
  ],
})
```
