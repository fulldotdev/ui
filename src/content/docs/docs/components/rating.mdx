## Installation

To install the rating component, run the following command:

```bash
pnpm dlx shadcn@latest add @fulldev-ui/rating
```

## Usage

```ts
import { Rating } from "@/components/ui/rating"
```

```astro
<Rating rating={4.5} />
```

## Examples
