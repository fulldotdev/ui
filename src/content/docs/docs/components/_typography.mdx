## Installation

To install the typography component, run the following command:

```bash
pnpm dlx shadcn@latest add @fulldev-ui/typography
```

## Usage

```ts
import { Typography } from "@/components/ui/typography"
```

```astro
<Typography>Text content</Typography>
```
