## Installation

To install the banner component, run the following command:

```bash
npx shadcn@latest add @fulldev/banner
```

## Usage

```ts
import { Banner, BannerContent } from "@/components/ui/banner"
```

```astro
<Banner>
  <BannerContent>
    <BannerTitle> Banner Title </BannerTitle>
    <BannerDescription> Banner Description </BannerDescription>
  </BannerContent>
</Banner>
```
