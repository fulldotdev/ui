## Installation

To install the badge component, run the following command:

```bash
pnpm dlx shadcn@latest add @fulldev-ui/badge
```

## Usage

```ts
import { Badge } from "@/components/ui/badge"
```

```astro
<Badge variant="default">Badge</Badge>
```

### Link

You can use the `href` attribute to render the badge as a link element.

```astro
---
import { Badge } from "@/components/ui/badge"
---

<Badge href="/">Badge</Badge>
```
