## Installation

To install the form component, run the following command:

```bash
npx shadcn@latest add @fulldev/auto-form
```

## Usage

```ts
import { AutoForm } from "@/components/ui/auto-form"
```

```astro
<AutoForm fields={[{ label: "Name", type: "text" }]} submit="Submit" />
```
