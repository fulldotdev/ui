## Installation

To install the tile component, run the following command:

```bash
pnpm dlx shadcn@latest add @fulldev-ui/tile
```

## Usage

```ts
import { Tile, TileDescription, TileTitle } from "@/components/ui/tile"
```

```astro
<Tile>
  <TileTitle>Title</TileTitle>
  <TileDescription>Description</TileDescription>
</Tile>
```
