## Installation

To install the logo component, run the following command:

```bash
pnpm dlx shadcn@latest add @fulldev-ui/logo
```

## Usage

```ts
import { Logo, LogoImage, LogoText } from "@/components/ui/logo"
```

```astro
<Logo>
  <LogoImage src="logo.svg" alt="Logo" />
  <LogoText>Brand Name</LogoText>
```
