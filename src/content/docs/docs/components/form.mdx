## Installation

To install the form component, run the following command:

```bash
pnpm dlx shadcn@latest add @fulldev-ui/form
```

## Usage

```ts
import { Form } from "@/components/ui/form"
```

```astro
<Form fields={[{ label: "Name", type: "text" }]} submit="Submit" />
```
