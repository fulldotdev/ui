```astro live
---
import placeholderImage from "@/assets/placeholder.webp"

import { Button } from "@/components/ui/button"
import { Image } from "@/components/ui/image"
import {
  Section,
  SectionActions,
  SectionContent,
  SectionProse,
} from "@/components/ui/section"
---

<Section>
  <SectionContent>
    <SectionProse>
      <h2>Section heading</h2>
      <p>This is a section component for organizing page content.</p>
    </SectionProse>
    <SectionActions>
      <Button>Action</Button>
      <Button variant="outline">Action</Button>
    </SectionActions>
    <Image class="rounded-lg" src={placeholderImage} alt="Section Image" />
  </SectionContent>
</Section>
```

## Installation

To install the section component, run the following command:

```bash
pnpm dlx shadcn@latest add @fulldev-ui/section
```

## Usage

For elaborate examples, see the [blocks](/blocks) page.

## Examples

### Default

```astro live
---
import { Section, SectionContent, SectionProse } from "@/components/ui/section"
---

<Section>
  <SectionContent>
    <SectionProse>
      <h3>Section heading</h3>
      <p>This is a section component for organizing page content.</p>
    </SectionProse>
  </SectionContent>
</Section>
```

    <SectionProse>

    </SectionProse>
  </SectionContent>
