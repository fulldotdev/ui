---
title: Section
description: A layout component for organizing page content into sections.
---

For elaborate examples, see the [blocks](/blocks) page.

```astro live
---
import { Button } from "@/components/ui/button"
import { Image } from "@/components/ui/image"
import {
  Section,
  SectionActions,
  SectionContent,
  SectionGroup,
} from "@/components/ui/section"
import { Typography } from "@/components/ui/typography"
---

<Section>
  <SectionContent>
    <SectionGroup>
      <Typography>
        <h2>Section heading</h2>
        <p>This is a section component for organizing page content.</p>
      </Typography>
      <SectionActions>
        <Button>Action</Button>
        <Button variant="outline">Action</Button>
      </SectionActions>
    </SectionGroup>
    <Image
      class="rounded-lg"
      src="/src/assets/placeholder.webp"
      alt="Section Image"
    />
  </SectionContent>
</Section>
```

## Installation

To install the section component, run the following command:

```bash
pnpm dlx shadcn@latest add @fulldev-ui/section
```

## Usage

For elaborate examples, see the [blocks](/blocks) page.
