## Installation

To install the section component, run the following command:

```bash
pnpm dlx shadcn@latest add @fulldev-ui/section
```

## Usage

```ts
import { Section, SectionContent } from "@/components/ui/section"
```

```astro
<Section>
  <SectionContent>Content</SectionContent>
</Section>
```
