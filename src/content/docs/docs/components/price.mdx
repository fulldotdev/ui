---
title: Price
description: A flexible price display component for showing pricing information.
---

```astro live
---
import { Price, PriceValue, PriceUnit } from "@/components/ui/price"
---

<Price>
  <PriceUnit>$</PriceUnit>
  <PriceValue>99</PriceValue>
  <span class="text-sm text-muted-foreground">/month</span>
</Price>
```

## Installation

To install the price component, run the following command:

```bash
pnpm dlx shadcn@latest add @fulldev-ui/price
```

## Usage

```ts
import { Price, PriceValue, PriceUnit } from "@/components/ui/price"
```

```astro
<Price>
  <PriceUnit>$</PriceUnit>
  <PriceValue>99</PriceValue>
  <span>/month</span>
</Price>
```
