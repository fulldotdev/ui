# Code

Enhanced code component with live preview.

```astro live
<h1>I'm a live preview</h1>
```

## Usage in html

Use it in the same way as the native Astro `<Code/>` component, which is documented [here](https://docs.astro.build/en/reference/components-reference/#code-).

The following code will translate to the example at the top of this page:

```astro
---
import Code from 'fulldev-ui/components/Code.astro'
---

<Code code={`<h1>I'm a live preview</h1>`} />
```

Will render as:

<Code code={`<h1>I'm a live preview</h1>`} />

## Usage in markdown

Thanks to [this great plugin](https://github.com/mattjennings/astro-live-code) you can also use it directly in markdown files.

The following code will translate to the example at the top of this page:

````html
```astro live
<h1>I'm a live preview</h1>
```
````
